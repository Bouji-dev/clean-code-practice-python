# clean-code-practice-python
A simple CLI TODO List application in Python to practice and demonstrate Clean Code principles from Robert C. Martin's book (Uncle Bob). Step-by-step refactoring journey: meaningful names, small functions, comments, formatting, error handling, SOLID principles, and more. Perfect for learning clean, readable, maintainable Python code. #CleanCode
